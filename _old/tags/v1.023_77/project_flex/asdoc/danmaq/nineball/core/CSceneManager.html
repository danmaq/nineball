Packagedanmaq.nineball.core
Classpublic final class CSceneManager

シーン進行を管理するクラスです。

このクラスにシーンを登録し、そしてこのクラスを通じ実行させます。 複数シーンのスタックを積むことも出来ます。 (この場合、一番若いシーンが実行されます)

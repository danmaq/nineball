状態による制御AIを持った表示オブジェクト。

AIであるcontextオブジェクトには、毎フレーム開始時にupdate() メソッドが呼び出されます。このオブジェクトに任意の状態を設定して制御します。 状態からはcontext.ownerで表示オブジェクトを参照することができます。

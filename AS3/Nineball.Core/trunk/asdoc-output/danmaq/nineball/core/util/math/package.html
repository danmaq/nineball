public function contains(expr:Number, limit1:Number, limit2:Number):Boolean

値が範囲内に含まれているかどうかを取得します。

limit1に最小値、limit2に最大値を設定することを推奨します。 逆にしても内部で自動認識して対応しますが、余計な負荷がかかります。

Parameters

expr:Number — 対象値。
 
limit1:Number — 限界値。
 
limit2:Number — 限界値。

Returns
Boolean — 値が範囲内に含まれている場合、true。

wrap()function 
public function wrap(expr:Number, min:Number, max:Number, allowMin:Boolean = true, allowMax:Boolean = true):Number

値を指定された範囲内に制限し、超過した場合は範囲内でループします。

注意：最小値と最大値を逆さに設定しても内部で自動的に認識・交換しますが、 無駄なオーバーヘッドが増えるだけなので極力避けてください。

Parameters

expr:Number — 対象の値。
 
min:Number — 制限値1(最小)。
 
max:Number — 制限値1(最大)。
 
allowMin:Boolean (default = true) — 制限値1(最小)と同値の状態を許容するかどうか。
 
allowMax:Boolean (default = true) — 制限値2(最大)と同値の状態を許容するかどうか。

Returns
Number — 制限された値。
